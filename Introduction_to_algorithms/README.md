## 📖 The Book

**Grokking Algorithms** by Aditya Bhargava is a beginner-friendly, illustrated guide to algorithms and data structures. It breaks down complex computer science concepts into simple, visual explanations that anyone can understand.


## 🧠 What I Am Learning

Algorithms are step-by-step instructions for solving problems. This book teaches me how to:

- **Search** data efficiently
- **Sort** data in the right order
- **Understand Big O notation** — measuring how fast an algorithm is
- **Think like a programmer** — breaking problems into smaller pieces
